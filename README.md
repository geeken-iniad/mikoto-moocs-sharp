# Mikoto (MOOCs Sharp): MOOCsを便利に

Mikotoは、MOOCsをより便利に利用するためのブラウザ拡張機能です。

## 機能

[プロジェクトボード](https://github.com/orgs/iniad-mdx/projects/2)

- [x] ⚠️ 出席、課題があるタブを色付けして目立たせる
- [ ] 🕰️ 今・次の授業コマ情報の表示
- [ ] 📅 時間割の表示
- [x] ✨ デザインをより見やすく
- [ ] 🌙 テーマ切り替え (ダークテーマなど)
- [x] 👆 講義カードをクリック可能に
- [x] 📏 サイドバーを大きくし、クリックしやすいように
- [x] 🔢 文字数カウンター
- [x] ⌨️ Ctrl + Enterでフォーム提出
- [x] 🔁 数字キーでタブを切り替え
